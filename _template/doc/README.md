# images/

Put this project's screenshots and diagrams here — evidence for PIM
activations, Conditional Access policies, pipeline runs, dashboards,
before/after comparisons, etc.

**Naming convention:** `NN-short-description.png`, e.g. `01-pim-eligible-config.png`,
`02-approval-workflow.png`. The number prefix keeps them in the order you'd
walk someone through the project, and makes them easy to find later when
prepping for an interview.

**Referencing an image from this folder's `index.html`:**

```html
<img src="images/01-pim-eligible-config.png" alt="Describe what the screenshot shows">
```

No `../` needed — the image and the HTML file are in the same project folder.
